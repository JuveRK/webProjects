Latest Project: Wanderlust Dance

Posted September 26, 2013

This is the initial posting of my latest project: a professional dance website for my girlfriend. The site is built with Ember.js so that blog posts and shows can be added to their respective JSON arrays and the site will automatically be updated. It also features an Ember.js helper that creates Google Maps iframes from just an address and place name. Currently the site is using ember-data to store the models, but I am working on building a backend for data persistence using jsonengine.

todo for Wanderlust Dance:
  • Finish contact page
  • Finish customizing jsonengine for my particular data persistence needs
  • Rewrite Ember app to use jsonengine
  • Build ember.js Admin app for managing blog posts and the show list, this will be deployed on Google's App Engine alongside jsonengine
  • Find photo gallery storage solution, integrate with Admin app

Eventually I'd like to write node.js and LAMP stack backends for this site as well, but I think jsonengine will be enough to run this site until it gets more traffic. As always comments are highly appreciated, just shoot me an email.
